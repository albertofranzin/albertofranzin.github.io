Human-AI interaction: ARIAC WP1

This page contains some general information about the activities of the ARIAC WP1 on Human-AI interaction, and its five main tasks:

  1. Human-in-the-loop
  2. AI-in-the-loop
  3. Consensus mechanisms
  4. Multiple and imperfect annotations
  5. Explainable AI
This work package is animated by Prof. Christine Decaestecker and Alberto Franzin (LISA, ULB).
